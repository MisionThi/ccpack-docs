# Armor json structure

The sturcture for all armor pieces is the same but it depends on in which folder it is to choose the armor piece.

```json
{
	"namespace": "example_pack",
	"id": "emerald_boots",
	"name": "emerald",
	"durability": 47,
	"protection": 0,
	"enchantability": 0,
	"toughness": 0,
	"knockback_resistance": 0
}

```
